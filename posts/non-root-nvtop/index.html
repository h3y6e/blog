nvtopを非rootでインストール

2020-01-28
#memo

htop の nvidia-smi 版的なやつ： nvtop
いい感じになるのでオススメ。

$ git clone https://github.com/Syllo/nvtop.git
$ cd nvtop
$ cmake -DCMAKE_INSTALL_PREFIX=$HOME/local
$ make
$ make install

その後 .zshrc 等に以下を書いて反映させる。

export PATH=$PATH:$HOME/local/bin
